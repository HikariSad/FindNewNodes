## 简介

该项目旨在从提供免费节点（代理，VPN等）的网站上爬取数据，并将数据编译成方便使用的格式，可用于其他应用程序或个人使用。

## 功能

- 爬取多个来源的免费节点
- 以结构化格式输出节点（例如JSON，CSV）
- 可配置的爬取间隔
- 易于扩展以添加其他来源
